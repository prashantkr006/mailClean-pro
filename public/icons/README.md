# Icon Files Required

This directory needs the following PNG icon files for the Chrome Extension:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels  
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Design Guidelines

- Clean, professional design
- Use the MailClean Pro brand colors
- Simple geometric shapes or envelope/mail icons
- Consistent across all sizes
- Transparent background

## Generation

You can create these icons using:
- Figma
- Adobe Illustrator
- Online icon generators
- Or commission a designer

## File Naming

Ensure files are named exactly as above and placed in this `public/icons/` directory.
